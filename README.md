# DataCopilot

An interactive statistical assistant designed to help non-technical users perform exploratory data analysis without requiring programming knowledge.

DataCopilot automates common statistical workflows such as descriptive analysis, correlation analysis and simple linear regression while providing accessible interpretations of the results.

---

## Overview

Many people need to analyze data but lack formal training in statistics or programming.

DataCopilot was created to simplify introductory statistical analysis and make analytical workflows more accessible through an interactive and beginner-friendly interface.

The project focuses on:
- accessibility
- statistical interpretation
- educational support
- automation of basic analytical tasks

---

## Features

- Descriptive statistical analysis
- Correlation analysis
- Simple linear regression
- Automated interpretation of outputs
- Interactive analytical workflow
- Beginner-friendly interface
- Statistical summaries for non-technical users

---

## Current Capabilities

### Descriptive Statistics
Generate summary statistics for datasets, including:
- mean
- median
- standard deviation
- minimum and maximum values
- distribution summaries

### Correlation Analysis
Evaluate relationships between variables using correlation metrics and basic interpretation of results.

### Simple Linear Regression
Perform simple linear regression and automatically interpret:
- coefficients
- model fit
- relationship direction
- explanatory power

---

## Motivation

Statistical analysis is often inaccessible to people without technical backgrounds.

Many beginners struggle with:
- programming requirements
- interpreting outputs
- understanding statistical concepts
- organizing exploratory analyses

DataCopilot was developed as an educational and analytical support tool to reduce these barriers and help users interact with data more intuitively.
